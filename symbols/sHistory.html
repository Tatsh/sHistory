<static> sHistory.addEventListener(func)
Add an event listener. These are called every time the hash changes, which means that if one key changes but another state stays the same, then all listeners will still be called.

If you want to prevent the callback from running repetitively, save the last state value outside of the function and make sure it is different within the callback before the callback can continue.

{function()} func
Callback.
